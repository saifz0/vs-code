# Leaderboard

> The leaderboard website displays scores submitted by different players. It also allows you to submit your score. All data is preserved thanks to the external Leaderboard API service.

## Built With

- Html
- Csss
- Js
- Technologies used:
  > webpack
  > AJAX
  > API
  > Gitflow

```bash

```

Install with yarn:

```bash
yarn add webpack --dev
```
